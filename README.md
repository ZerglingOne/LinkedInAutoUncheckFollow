First, you must install TamperMonkey

https://www.tampermonkey.net/

Then open the extension with the puzzle piece icon in the top-right and go to the Dashboard.

Create a new script by clicking the + tab on the UI next to Installed User Scripts, Settings, and Utilities

Paste the entirety of the code in unfollow.js, even the "comment" metadata at the top.

Click File -> Save

Then it should appear in your "Installed Userscripts" tab, and should have a slider to activate/deactivate. Make sure the bar is green and you're good to go!

No more annoying unclicking the "Follow <Company> to stay updated!"
